## Getting Started

1. **Clone the Repository:**

```bash
 git clone https://github.com/YourUsername/JeffersonCC.git
```

2. **Navigate to a Course Directory:**

```bash
 cd JeffersonCC/CIS116_C++
```

3. **Compile & Run Code:** (Example for C++)

```bash
g++ main.cpp -o main
./main
```
